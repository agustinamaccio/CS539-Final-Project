Spotibility! CS539 Project

By Tess, Gabrielle, Agustina, Uday, Adityavikram

What is Spotibility?

We are proposing a tool to generate ‘friendships’ based on musical preferences and playlist composition of Spotify users. The tool will take a user’s Spotify playlist(s) as input and create matches with other Spotify users based on an algorithmic compatibility of the audio features of songs within each playlist. We are just students working on Machine Learning Class Project, we are not associated with Spotify.
